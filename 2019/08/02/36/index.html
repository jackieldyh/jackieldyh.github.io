环境不能带给我们什么

发表于 2019-08-02 更新于 2021-01-26 分类于 Travel 阅读次数： Valine：
本文字数： 1.7k 阅读时长 ≈ 2 分钟
2019暑期婺源学工学农纪行第2天

  环境本身并不能给我们提供任何东西，那么什么能够成为给予的来源？
　　我想真实的感受，是真实的感动来源的重要方面。
　　或许之前的准备再多，还是抵挡不住真实自有的万钧之力。
　　但是我们只是知道。我们目前所做的一切，都是有意义的，哪怕我们并不能说清楚，那种意义究竟是什么。
　　在连绵的大山和无尽的森林之中，有一种独属于生活的力量，它是如此地稀缺，对城市人而言，但是又是如此的平常，能够拼凑出真实的生命意义。
　　那啊很多时候，我们会被太多可以量化的东西牵绊，例如要交的材料，以及随之而来的，揣度老师喜好的过程。
　　但为什么不相信，一切都是最好的安排呢？
　　我们所做的一切，在以一种自我利益为中心的前提下，往往会让人失去思考很多外部性的能力。
　　徜徉在这种环境之中。尽管我们明知我们只是匆匆过客，但是我们所希望收获的，或许最为简单的，就是在这里，将时光浪掷。
　　很多时候，没有意义，就是最好的意义。
　　我们或许不能将未来的不确定性消减到零，但是我们至少可以对不确定性满怀期待和准备，取用一种更为平常的视角去看待，去感受，去经历。
　　一切都会是最好的安排，身处于这个环境，尽管这个环境不会让你对论语突然有着见识的增长，但是对于论语而言，从家族的角度，确实能够反映出很多现实与幻象相接触的地方。
　　哪怕有很多时候，我们并不能抓住重点，说一些漂亮话，写一些漂亮的文章。但是只是这种经历，就或许已经够用。
　　仿佛是在流水线上，保留着自己个性的尝试。
　　我们终究会要被送到远方，但是远方不一定是我们的宿命。
　　但是我们也总是要走向远方。
　　在旁人艳羡的目光之下，我们还是要清楚的知道，我们是谁，我们为何而来，我们要向何而去。
　　条件，或者说安排，一切都会是最好的。

  我是萌萌哒的分割线
　　以上或许是看到一些留言的直观感受

  今天所做的，确实有点像度假，毕竟，只是入住了民宿，然后形式主义地开了个会，随后的时间，都是在村庄漫游。
　　没有人追问着你要结果呀，只是你要知道，你不是来这里旅游的。
　　优渥的条件，空心的樟树，不被人经常使用的村委会还有为村小组范围都能发生争执的村民，构成了这个村庄的基本图景。
　　在这个村庄漫游，一条道路从村头延伸到村尾，两边都还是古旧的，带有浓郁地方特色的房子（或者更为准确的说，是黑白相间，具有马头墙的所谓的“徽派建筑”。走进房屋，还有原始的天井和重叠的阁楼。但是似乎这里的房子是没有正门的，门前似乎还残存着“家训”之类的吉利话。有些时候，这种诡异的统一和文化地残存，令人百感交集）。一条小河在村庄中间流过，划分了河东和河西。房屋旁大多有或大或小的田地，村庄的两头，靠近马路的田地是最多的，山坡上能看到开垦过的阶梯痕迹，但是明显没有种植农作物，倒是隐约能见到茶叶的影子。
　　阡陌交通，鸡犬相闻，或许是对这里的第一印象了吧。

  沿着主路旁的一条小路上山，两边的农作物对于我们而言是陌生的，只有一季的稻米正是抽穗的时候，农田里还有着稂莠的痕迹，辣椒有些已经结果，在绿油油的一片中格外亮眼。间或还有一些番薯、大豆和丝瓜、南瓜。
　　为什们能够认识它们呢？还是要借助着手机的识图功能。
　　大概是一种最为深刻的讽刺了吧。
　　拾阶而上，路旁总是有溪流潺潺，对坐的凉亭，停滞着老人们的时光。重门深锁，越向上路越窄，也逐渐荒芜。旁边出现了无人的坟冢，在这个应该是殡葬改革的年代，还是有些格格不入。面前还略显新鲜的祭品，提示着我们一种生机的痕迹。红土漫漫，小路戛然而止，回首一望，村庄炊烟袅袅。
　　小路旁，偶尔还会杀出一条狂犬，提醒着人们，这里毕竟不是你的家园。运送建筑材料的小车也时时经过，建筑不同的风霜，经历着时光的刷洗。

  一切是那样静谧，公路的车行很少。
　　夜晚那明亮的星河，间或闪烁着飞机的身影。
　　北极星低悬在山头，重重的山林中，似乎有隐约的灯光。
　　村子是黑暗的海洋。
　　第二天该会醒的很早吧。
August 2, 2019
