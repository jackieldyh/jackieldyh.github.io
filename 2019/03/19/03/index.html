来自过去的快乐

发表于 2019-03-19 更新于 2021-01-26 分类于 Daily 阅读次数： Valine：
本文字数： 1.5k 阅读时长 ≈ 1 分钟

Sentence

你与众不同，却希望与别人一样。
我觉得，这可称得上是一种恶疾了。
——保罗·科埃略《维罗妮卡决定去死》

Poem

赤枣子·寄语酿花风日好
[清]纳兰性德
惊晓漏，护春眠。格外娇慵只自怜。寄语酿花风日好，绿窗来与上琴弦。

  春色正好，这首小令也是写出了春色与春愁。往往最美好的事物总是要来一些它的反面才能显示出它的真实性。平凡的生活总是需要有一些波折，但看着窗外的美景也仿佛有了希望。忧郁的时光总会过去，而过去之后还能全身而退，控制后自己的欲望与索求，明白自己从何处来，到何处去，也是一种智慧吧。

Excerpt

  社会学把人视为社会人，而非自足的个体；它不是把每个人视为抽象人性的一个特殊现身(incarnation particuliére de l’humanité abstraite)，而是把他视为从一个特殊的集体人性即社会中冒出来的一个多少具有自主性的芽儿(un point d’emergence plus ou moins autonome d’une humanité collective particuliére d’une société)。这种观看个体的方式，如果要在一个个体注意的宇宙(l’univers individualiste)中具有现实性的话，就不得不采取体验、甚至个体启示的形式，这也是为什么我会称其为“社会学的统觉(aperception sociologique)”。因此，年轻的马克思才会带着新手的夸张口气写道：“正是社会在我体内思考。”
杜蒙：《阶序人》

  连续两天选择了《社会理论》课的阅读材料，既是一种补充，也是一种纪念，是一种认知不断被挑战，被洗礼，去看到另外一个世界的过程。在这个过程之中，正是因为有阅读，有思考，才能看到世界的另一面，能看到在现在看上去似乎是贫乏的语言背后的思想闪光。很多我们看似奇怪、看似绝对的表述背后，或许应该想一想，为什么其会成为经典？当然，不是一种绝对崇拜的态度，而是一种基本的敬畏， 去还原到一种历史事实和人类的高度，去解读、去理解、去把握。相信这条路很长，也很枯燥，但是那一点点的明白与思考，或许就是这件事情的最大意义。也正是因此，才明白现实可能是最不重要，也是最浮躁的，只有将自己沉淀过，才会明白什么是黄金，什么又是沙粒。

Comments

  时间不一定是显得错乱，我们认识的可能性和方法不一样。从伽利略变换到洛伦兹变换，在世界观的改变之下，是一种对世界统一和本源的更为深刻的认识。而在这个过程之中，我们不可能是独立存在的。尽管我们不会接受对个体过强的假设以至于个体成为一个一个的原子（或许那“精致的利己主义者”就是这样产生），但同样社会的阴影也不应该时刻盘旋于每个个体的头顶。这或许矛盾，但也正是在这种微妙的平衡中，我们不断地认识世界。或许也正是这个阶段，我们才能够清醒的去想一些远大到甚至可能与事实无关的事情。一切都是飞鸿雪泥，但就是在每一瞬间中我们看到世界的模样。只有深刻明白这种暂时性，才能对世界有更大的感激。
　　也因此，在面对一切都不是理所当然的环境中，我们会去希望一种确定性。而世界本身的不确定性带来的是另一种形式的确定。我们会希望去切断一切形式的联系，但最终发现我们不可避免的紧紧相拥。不可能总是活在自己的世界里，但是我们却可以活在一个理想的世界里。这两个世界， 也会让我们感到一种确定。就像在夕阳下的奔跑与落日后的奔跑永远感到不同，那种简单的快乐，在羽毛球拍的挥动之间，仿佛让人回到了高中。
　　在香樟之下，我看着晚霞。

March 17-18, 2019
