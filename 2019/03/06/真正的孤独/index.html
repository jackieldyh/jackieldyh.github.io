真正的孤独是生活在一群无法让你做自己的好人中间 ——伊迪丝·牛顿
# Poem
蝶恋花·出塞 [清]纳兰性德 今古河山无定据，画角声中，牧马频来去。满目荒凉谁可语？西风吹老丹枫树。 从来幽怨应无数？铁马金戈，青冢黄昏路。一往情深深几许？深山夕照深秋雨。
从理想到现实，很多事情不是自己能够决定的。国家的辉煌会转瞬即逝，人也是。 # Excerpt 　　苏格拉底，凡是稍微有一点儿头脑的人，在每件事情开始的时候总要求助于神，无论这件事情是大是小；我们也不例外，如果我们不是完全丧失理智的话，要想讨论宇宙的本性，考察它的起源，或者要是没有起源的话，它是如何存在的，我们当然也必须向男女众神求助，祈求我们所说的话首先能够得到诸神的首肯，其次也能为你所接受。” 　　——柏拉图《蒂迈欧》27c
　　从神到人，从理智到感情，从和谐到混乱，观念的转变，背后蕴藏的是对这个世界无限的好奇。没有必要谁一定要说服谁，只是有理有据有节地思考与讨论就好。 # Comments 　　我们不能总是满足于惊叹，不能满足学到的只一点点看上去像是信息的知识。 　　我们需要选择思考，但思考是需要时间的，需要精力的。 　　简而言之，思考是累的，是不一定被他人认可的。 　　但不论如何，思考是个好东西，我们总是要拥有它。 March 6,2019 文中未标明的图片均为作者自摄 ，如需转载请在下方评论区留言或使用邮箱
