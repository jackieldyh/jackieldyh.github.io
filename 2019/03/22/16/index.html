阳光正好，岁月呢？

发表于 2019-03-22 更新于 2021-01-26 分类于 Daily 阅读次数： Valine：
本文字数： 1.4k 阅读时长 ≈ 1 分钟

Sentence

春分是夜晚和白昼的和解。

Poem

《回乡偶书二首》（其二）
[唐]贺知章
离别家乡岁月多，近来人事半消磨。
惟有门前镜湖水，春风不改旧时波。

  一首非常短小，而又非常浅显的诗句，就像我们熟知的另一首诗一样，“少小离家老大回，乡音未改鬓毛衰。儿童相见不相识，笑问客从何处来。”小学时候咿呀学语的日子，如今早已远去，我们再也不会感受到那样青涩的时光。世界对我们打开，但却从来没有提醒我们他有多么复杂。我们在池子里泅渡，却像鱼一样只拥有七秒钟的记忆。我们随水流飘荡，去总是要回到自己来时的路。可是有多少人真正的如此幸运，能够走上那条路呢？或许只有镜湖水，关山月，无语凝噎。

Excerpt

  金融和经济危机的发生是资本主义制度的本质特征之一。工业革命以来，资本主义世界危机频繁发生，20世纪30年代大萧条和本次国际金融危机是其中蔓延最广、破坏力最大的两次，他们都是资本主义内生矛盾积累到无法自我调节程度后的集中爆发。
　　——刘鹤《两次全球大危机的比较研究》

  党课小组读书会的一本书，我之前就说过其实在这学期并不准备去阅读一些非常现实的书，但是，这种书也有着自己的吸引力，能让我从对哲学盲目的思考中解放出来，让人有一种成功的快感和站在地面上的踏实。我们总是会有两个世界，这两个世界相互交错，相互联系，才能使我们成为一个完整而有生命力的人。
　　而这与这本书的具体内容，坦率的来说，我是并不认为他值得这本书的定价的。七篇报告的堆砌，加上并不亲民的表述，让这本书几乎能够成为一本需要众多二手文献才能真正理解的书。与此同时，书中的内容也多有重复，感觉仿佛是高中政治和历史教材的简单叠加，正确的事情说三遍。

Comments

  又是一天的朝阳，听着文图的开馆曲，就知道这一天又要开始了。而我们需要想的是，一天一天，究竟是时间的刻度， 还还是能够容纳更丰富的内涵？我们总是会有时刻很‘丧“，但世界又要求我们不能轻易表达。这个世界太冷漠而又太过敏感，是对自由的深刻限制和怼理想的无情嘲弄。总是要有个港湾，而这个港湾最好能让我们仰望星空。例如里克尔，例如历史，例如宗教。
　　这并不是说要真正的去信仰，而是要明白自己的问题，曾经在历史上被那么多的人思考过，形成了一个又一个封闭的体系。我并不想那么快的失去自由，但是也必须要明白这些事情原有的思路。我们所面对的世界是独一无二的，与此同时也是最令人觉得浅薄与丧气的。我们或许只有跳脱出来，才能看到这个世界被人遗忘的一些美好。
　　那美好太过柔弱，静静的在远处欣赏就好，不要去消费它。
　　或许越来越习惯用”我“而不是”我们“，那一种集体归属感的疏离，大概也是无法挽回的吧。终于知道，人生路真的是只能供一个人行走的单行道，一切美好都是如此虚幻，而一切痛苦又只能由自己承担。看着远处忙忙碌碌的人们，或许此刻的小幸福，就是最能让人感到快乐的事情了吧。天边的朝阳与夜晚的月亮都是如此明媚，而只有在特定的好时节，才能发现她的美丽。
　　何处无月，何处无竹柏？

  或许这里越来越像某个树洞了吧。一种表达的欲望促使着人在这上面留下点什么。他可能是负能量，可能是一些浅薄的句子，但是这些又有什么不好呢？总是要找到一个发泄的渠道，去在写作中认清自己的位置，知道自己的方向，不是吗？
　　人或许就是在这样一次一次的试错中成长的。
