北京的天是灰蓝的

Posted on 2019-04-05 | In Comments | | Visitors:

　　“经天纬地才能，由学问成就”
　　究竟意味着什么？一种精神，如何支撑着人一直前进？
　　或许可以把它解释成，一种信念，或者是，一种期望。
　　尽管并不想活在期望里，活成自己不想看到的样子。
　　但总是要屈服于现实。

　　那些过去的回忆，或许只有通过描述，才能渐渐变得真实。最初的梦想，被遗忘在角落，裹紧了灰尘。
　　是不想擦干，还是不敢呢？
　　那些最淳朴的快乐，那些单纯的甚至有些想笑的日子，大概是一去不复返了吧。
　　我从时光打马走过，留下孑孓的背影。
　　总是在个体化、原子化的道路与渴望得到团体认同中游弋。

　　会否会害怕，最后两个场合都不能融入呢？
　　在摇晃的未来，总是要去抓住一些确定性，或者，是听闻同样的遭逢。
　　来到北京的第九个月。

　　最终总是要去做出选择，不能总是将自己视作一个特殊的存在。人类知识的海洋，过于深广，却总是要从最为微茫的地方开始，一步一个脚印。
　　直到最后，都会感到前路漫漫。
　　口口声声说的不功利，最后会去向何方呢？
　　不在意别人的目光，或许是一件单纯，而不简单的事情吧。
　　毕竟从某个时间开始，就已经活在他人的目光下了

　　家家争唱饮水词，纳兰心事有谁知。
　　或许换了任意一个人名，这句话都能成立吧。

　　那些熟悉的名字，最终会活在单纯的记忆里，去往另一个世界。在那有限的交际中，究竟应该用怎样的方式，去勾起那一段段的记忆呢？
　　所谓的“聚是一团火，散若满天星”。
　　原谅我永远不懂这句话的褒贬。
　　灰蓝的天幕下，杨絮肆意的挥洒，在能感受到的泥土中，成为肇事的元凶。
　　那一朵朵小小的花儿， 在称羡的眼光中，开了又落。
　　春去春又来。
　　但没有一个春天会再像这个春天了。

April 5,2019
