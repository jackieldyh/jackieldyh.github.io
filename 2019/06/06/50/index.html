我多害怕，在雨中奔跑

发表于 2019-06-06 更新于 2021-01-26 分类于 Comments 阅读次数： Valine：
本文字数： 1.6k 阅读时长 ≈ 1 分钟

如果真的是像博文下所标明的那一天的话，那么全国大部分地区（为什么感觉这个定语这么熟悉？）的考生应该都考完了高考，更确切的说，因为今年政策的变化，更多的人需要来参加自主招生，自主招生在经历了风风雨雨之后，今年可谓是一个“小年”，但只要有实力，有积累，形式永远只是发挥的空间。
高考加油!

  第一次淋到这么大的雨。
　　或许是在高中习惯了不带伞的日子，无论怎样的雨都能够坦然的在路上行走，甚至感觉，那小雨淅淅沥沥的感觉，是一种净化的救赎。
　　现在来看，当时无忧无虑的样子，该是很美的吧。
　　至少我们总是在失去之后才知道去珍惜。

  其实只要肯抬头望望天，就能够发现这场雨来之前已经是“山雨欲来风满楼”，那阴沉的乌云昏黑了整个天幕，让本来燥热的溽暑，增添了几分阴森的气息。
　　人而我还不信，也不料，竟会变成一场由雷声开场，进而忽然之间下大的淅沥。
　　看来我还是对这个城市太不了解，或者说，不太敢去接受皇天后土给予我们的现实。

  雨开始的几滴，是让人舒心的，一开始，总是让人有一种“斜风细雨不须归”的错觉，但是这或许对于今天而言，是一场过于温柔的开场。很快的，雨滴就从细线变成豆大，打击声甚至能够谱成一首交响乐。但是这所学校是不能停止的，从小雨到大于，该奔跑的人还是在奔跑，自行车轮毂的转动，溅起了水珠。
　　我未曾想，短短的五百米距离，就算是有了自行车，还是会变得如此漫长，以至于到了最后的一百米，简直是在用一种本能去完成骑行，停车，走向寝室的动作。
　　很奇怪，走向寝室的那三十米，我忽然不再火急火燎，那或许是一种失败之后的自暴自弃，任凭雨滴越来越狂，任凭朔风逐渐停止，天地之间，似乎只剩下了雨滴，在做自由落体。
　　我第一次发现，原来雨滴真的可以模糊人的视线，像是模糊远处的景物一样，它在我们面前，连成了一道水幕，没有任何延迟，就融入了可以融入的任何东西。
　　对于生活而言，一些不好的食物，也是这样融入的吧。
　　让人就算想摆脱，也欲罢不能。

  很多东西，知道用数字表现出来，才会让人感受到一种刺骨的真实吧。
　　也只有一种能够量化的标准，尽管不近人情，但是却是最为公正而令人无可辩驳的。
　　但是我们是人，人也同样会要具有人性化的空间。
　　于是乎我们用着“最为先进”的系统，却只是用着这个系统记录下一些事实。
　　或许技术的力量，也只是提醒我们一些事实吧。

  说到底，还是要来说一说有关高考的事情。
　　尽管我们已经远去，像是在这个时代最为盛大的一场狂欢。
　　每个人都怀念着自己的青春岁月呀。
　　但是真正面对它的时候，你才会发现，现代社会的原子化特征。
　　就算全世界都为我保驾护航，但是能改变自己民运的，永远只有你自己。
　　这是属于中国的公平，相比起另一个人口大国。
　　同样，如果你是在“超级中学”就读的话，相信你也可以为你自己的前途放宽心，因为你至少可以相信你获得的三年教育，使你已经在潜移默化之中，能够去获取一个别人的上限要高的分数。
　　尽管你也不会知道这样的优势是怎么产生的。
　　但是这就是量化告诉我们的事实呀。
　　然后经历了这一场考试，你会发现，这三年其实真的太过easy，竟然把上限都给你写好，你还能够轻松的够到。
　　之后，我们的征途是星辰大海，只是我们再也不知到何处是上限。
　　我们永远透支着自己，假装自己已经努力到了无能为力。
　　与此同时，你还可以拥有你自己的生活，去回击这个社会上的偏见。
　　你会发现我们所接受到的信息，实在是太过片面，简直是一种刻板印象。
　　你也只能如鱼饮水，冷暖自知。

  在我们之后，又有一届的同学，成了“最好的上一届“
　　我想其他的都不用多说，两天，九个小时，是高中的谢幕，也是青春的涅槃。
　　你值得拥有一个看待世界不一样的视角
　　你也应该有这样一个视角。
　　愿你披戴荣光，你会笑着回头望。
June 9, 2019
