我仍会偶尔想起你

发表于 2019-06-06 更新于 2021-01-26 分类于 Comments 阅读次数： Valine：
本文字数： 509 阅读时长 ≈ 1 分钟

  有很多话想说，就一起说了吧。
　　一场最为盛大的怀念狂欢。
　　

  我们总是有一种倾向，去怀念自己曾经的辉煌。
　　毕竟它就像一个最为美好的故事。
　　如果还有共同的经历，能够产生代入感，那就更好了。
　　只是无论怎样描绘，记忆总是会衰减的，
　　哪怕是曾经所珍视的。
　　我们总是认为，一切都是理所当然的，只是失去了之后才知道美好。
　　这个世界上有太多无法控制的事物，我们所追求的的确定性，确实这个世界最没有未来的的活法。
　　我们终究要从历史中走出来，去面对这无比现实的人生
　　退一寸有退一寸的欢喜
　　理想主义只能存在于象牙塔中
　　但是这并不代表着我们不能带着这颗初心上路
　　现实的风霜，不改理想的模样
　　我们只是需要一种蛰伏
　　我们所拥有的已经是最美好的模样
　　不要躺在最好的地方睡觉。

  未名湖，博雅塔
　　清华园，光华楼
　　当地名不再仅仅是地名
　　生活便翻开了一页新的篇章
　　人生的每一页，都要用力的写下呀
　　拼合的记录，是一段最美的时光
　　我们在这之中跳跃，见到了当年模样
　　更重要的是，让我们一路向前。
　　不论结果如何，都是一段最美的时光
　　三沐三熏，百事顺遂
　　高考加油。
June 10, 2019
