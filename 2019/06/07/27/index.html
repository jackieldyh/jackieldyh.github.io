复恐匆匆说不尽

发表于 2019-06-07 更新于 2021-01-26 分类于 Comments 阅读次数： Valine：
本文字数： 1.4k 阅读时长 ≈ 1 分钟

　　笔者按：这篇文章还是一篇旧文，看起来还是一篇写在旅途上的文字。我想火车总是一个很神奇的空间，能让空间的距离感通过时间的延展抹去，同时通过一种对未来的深刻不确定性的思考，反思自己应该有的模样。不论是近乡情更怯，或是不敢问来人，之后的人生道路总是只能有自己去走，我们所拥有的反思，不应该仅仅是反思，更应该落实到反思之后的行动之中。
　　哦，我想起来了，这篇文章是写在今年寒假，与宇薇一起回雅礼的高铁上，那茫茫的大雾，确实可以迷失很多人的方向吧，尤其是距离高考不足100天的人身上，我们作为所谓的“学长学姐”，或许不能成为你们生命中的一道光，但是我们至少可以用我们极力发出的微光，去让自己不断的处于发光的过程之中。
　　在这个政治不正确已经成为政治正确的年代，拥有自己的判断与思考是那样的重要，而这些，只有“不畏浮云遮望眼，自缘身在最高层”。

  在这茫茫大雾中穿行于中原，忽地有一种迷幻的色彩。
　　阳光被空气里的尘埃阻挡，散射，留下，一个无力的白色身影。
　　广袤的农田被白色气息所淹没，近旁的公路伸向未知的远方。
　　车行被雾气吞没，终究被复兴甩下
　　路旁甚少有人家的烟火，那氤氲的雾气，仿佛提醒着个中人，今天也是一个节日
　　所谓近乡情更怯，更多是不敢面对自己心中“美好”的回忆
　　毕竟一旦对现实书写，就要迎接他的阴暗面。
　　一种奉献的激情蓄积在脑海，毕竟，从某种程度上，我也终于成为了之前梦想成为的一个人。
　　我不清楚，是否过于深刻的记忆，会一直鲜活
　　因为总是有人去让我们不由的面对现实，思考现实
　　曾经的高光时刻，会否永远停留在昨天？
　　祭奠一个回不去的过过往。
　　那些有趣的事，还记得吗？

  仿佛是种冥冥的对应
　　又是一场大雪，连北国也甚少经历的大学
　　更为砭骨的严寒提醒着人们生活的艰难
　　在这座城市里，我们正成为着过客
　　谁还记得两点之前，也是一班高铁，大雪让我们晚点了六个小时，再回到南方的空气之中，甚至感觉有些闷热
　　天气的温度，与情感也有密切的关系吧。
　　一步一遥。
　　我们曾经笃定地相信的，不变的，追寻的，被现实如此快的扭曲。
　　或者说，我们还没有看到下一个值得我们追寻的明确的目标
　　很多时候的生活，竟然陷入了一种随波逐流
　　我们如无力地苇草，在冬日的寒风中，寻找自己的根脉
　　或者说，发现自己，原来还是有着根脉的。
　　才发现，我们终究在故乡变成了过去时，属于现实的世界，早已换了模样。

  我们本可以知道更多，但是我们选择性地接受
　　我们与过往之间，究竟区隔在何方？
　　我们被急着被推向社会、承担责任，却不知，我们仍然希望拥有，或者说全无准备就要放弃，柔弱的权力。
　　车过洞庭，那南国罕见的雪正在消融，那密布的水网，暗示着南国的灵动。
　　但这都仅存于想象，漆黑的天幕没让我如七月般告别清秋，告别潇湘
　　湖南清绝地，万古一长嗟。
　　我们被囚于生活的不确定性之中，被空中的电波束缚，满足他人的期待。
　　那繁忙的交通线承载着无数人的梦想，只是哪怕一路同行，也最终渐行渐远。
　　会吗？
　　那一声长长的汽笛，是最长的哀鸣。
　　所谓的快乐，证据是一张虚伪的笑脸。
　　曾有的热情被现实消融
　　生活给予我们的，除了时空，还剩下什么？
　　祭奠一个关系日益稀薄，或者说，物质化的世界。
　　我最终会成为不一样的，令自己讨厌的人吧
June 11-12, 2019
