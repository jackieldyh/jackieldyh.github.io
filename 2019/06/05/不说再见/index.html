不说再见

Posted on 2019-06-05 | In Comments | | Visitors:
Words count in article: 958 | Reading time ≈ 3

　　   如果真的是博文下面写的时间的话，那大概你已经坐在雅礼窗明几净的教室里，享受着你三年以来最为宽敞的座位，期待着一张试卷的来临吧。
　　不论如何，还是要说一句：
　　高考加油!

　　  一个星期之前笔者在笔者按里写，大概这一个月不会保持更新的频率，但又怎奈这个月初有趣的事实在是太多，让人忍不住有一个地方能够抒发自己表达的欲望呢？
　　哪怕这些会是刚一开头就煞了尾。

　　  关于高考的回忆，如果真的要去书写的化，恐怕也被莫名其妙的格式化了。
　　我不知道是为什么，但是似乎就是单纯的，不希望自己再去回溯，只是知道，有一个不温不火的结局。
　　我总是愿意把高考，放在整个高三的语境中来看。
　　它仿佛只是这段时间必然要发生的一个句点，是那段时间里，唯一具有确定性的来源。
　　尽管我们为了这种确定性中的不确定性，已经给自己画了一个靶子，不断地瞄准。
　　但是瞄的越准，就越打不中呀。

　　  可能是因为那几天一直在校内，也没有让任何家人来，因此外界的纷扰对我来说，仿佛也是这个社会最大的一场狂欢。
　　一张张试卷写下答案，仿佛并没有什么不同，那种曾经期许的仪式感，并没有预期而至。
　　我只记得英语考完之后，那一抹最亮的阳光。
　　还记得那天中午才下了雨。
　　曾经的闷热被清爽所洗刷
　　我久久地不愿离去
　　怕离去了就再也回不来
　　我们终究回不去呀

　　  那两天每天中午和下午第一个走向食堂，在空空荡荡的食堂中，打上一份饭，在自己经常坐的那个座位，没有情感的吃完。
　　总是觉得那两天的饭菜，少了一些盐巴。
　　但或许我们也不需要更多的刺激了吧。每一天都是能决定历史的呀
　　尽管我们都不知道怎样去决定历史
　　往往人生的转向就是从这样的事实开始的吧
　　但是我们终究没我办法把握结局

　　  记得那棵香樟树下潘老师给每个人写下“加油”的样子
　　或是胡老师十分关心的文综究竟考了些什么
　　我大概是从来不介意别人会问我什么的
　　大概一切事物都会变成其最好的模样
　　我们要做的，或者所能做的，就是把自己能做的做到无能为力吧
　　尽管我们现在会发现，自己隐隐的离这样的优秀都越来越远。
　　追忆过去，或者说回忆过去，是为了对自己的现实不满找到一种发泄吧。

　　  但是我们仍然是对学弟学妹，或者说对雅礼有一种本能的情感
　　近乡情更怯
　　仿佛背负着一种压力，不能丢母校的脸
　　“母校”，居然还真的会有这样的情愫
　　我们希望这个世界变得更优秀
　　就从自己和他人开始
　　到那时我们总是首先需要把自己变得更强大

　　  说到底，高考只是一场考试
　　生活总是会要继续
　　未来终究会要到来
　　接受一个猩红的未来。
June 7, 2019
