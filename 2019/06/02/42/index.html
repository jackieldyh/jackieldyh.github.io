在资本面前，情怀或许只是笑话

发表于 2019-06-02 更新于 2021-01-26 分类于 Comments 阅读次数： Valine：
本文字数： 1.3k 阅读时长 ≈ 1 分钟

　　


对于美方发起的贸易战，中国早就表明态度：不愿打，但也不怕打，必要时不得不打。面对美国的软硬两手，中国也早已给出答案：谈，大门敞开；打，奉陪到底。经历了5000多年风风雨雨的中华民族，什么样的阵势没见过？

在实现民族复兴的伟大进程中，必然会有艰难险阻甚至惊涛骇浪。美国发起的对华贸易战，不过是中国发展进程中的一道坎儿，没什么大不了，中国必将坚定信心、迎难而上，化危为机，斗出一片新天地。

无论外部风云如何变幻，对中国来说，最重要的就是做好自己的事情，不断深化改革，扩大开放，实现经济高质量发展。美国下一步是要谈，还是要打，抑或是采取别的动作，中国都已备足了政策工具箱，做好了全面应对的准备。

这正如习近平主席所指出，中国经济是一片大海，而不是一个小池塘；狂风骤雨可以掀翻小池塘，但不能掀翻大海；经历了无数次狂风骤雨，大海依旧在那儿！
——2019年5月13日《新闻联播》国际锐评：《中国已做好全面应对的准备》

　　

  一晃贸易战也过去了一年多了。
　　或许从我的个人角度上看，是从一种被高考支配的恐惧，到了一种更为广阔的面向——甚至能够与文明的冲突和世界格局的扭转联系起来。
　　从这个意义上看，也是某种成长了吧。
　　也这是在这种认识不断更新之中，我们发现了之前所面对的信息的可笑，进而去思考我们所接受的一切的合理性。
　　这个世界，原来是如此的特别。

  但贸易战来的时候，甚至于在谈判之中，我们了解了太多的宣传——中国人永远彬彬有礼，嘴上挂着”合则两利，败则两伤“。
　　是呀，我们是礼仪之邦，我们是中华文明。
　　但是当文明之间对这种行为的体认发生根本性的差别的时候，我们就不由得应该思考，我们应该采用什么举动，或者是我们的思想，是不是应该更多的推己及人。
　　我们才发现，我们在自己的世界里活得太安逸，以为自己的世界就是整个世界。

  我不希望先入为主的认为“犯我中华者，虽远必诛”，人性，对于整个人类来说，都是差不多的。人们也很难在时间的积累中，收获处理人际关系的经验。
　　但是他就在那里，绕不过去。
　　嫉妒，愤怒是人之常情。
　　谁知道人们为了自己的利益，不论是精神上的，还是物质上的，可以做出怎样的行为呢？

  因此我们都应该更多些了解，就像这次刘欣与Trish的对话一样。
　　大肆宣传的背后，其实就是对数量太少的反思。
　　我们总是会在交流的时候寻找一些奇怪的怀胎，面对着一些奇怪的规定而不能理解。
　　最后还有些一些奇怪的收获。
　　仅仅把它变成一个过程不好么？
　　我们总是喜欢为事情赋予超出自身的意义。

  就像最近突然被回忆俘虏再打开摩尔庄园。
　　那绚丽的配色，熟悉的音乐，不能否认的是游戏的激励机制已经被奇怪的扭曲了。
　　我们面对的只是一堆的数字与虚拟的装饰
　　当这种氛围不再，一切的投入都只是徒劳。
　　我们只能去哀叹逝去的青春
　　毕竟，这一代青年人又有自己的玩法。
　　时光容易把人抛，红了樱桃，绿了芭蕉。

  但不论如何，游戏中不能成为金钱的俘虏，或者说，快乐不应该是用金钱买来的。
　　但是我们又要去哪里寻找原先的快乐呢？
　　今朝有酒今朝醉。
June 3, 2019
