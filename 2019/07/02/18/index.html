飞行-去看书中的世界

发表于 2019-07-02 更新于 2021-01-26 分类于 Travel 阅读次数： Valine：
本文字数： 843 阅读时长 ≈ 1 分钟

  这一次，也计划了很久的旅行吧。
　　我发现我自己，越来越习惯一个人撑伞，一个人走路，一个人看风景。
　　那本就不完美的表现欲，只有自己知道。
　　和这个世界保持距离。
　　

  这或许是我第一次踏上东北的土地，也是第一次我自己主导，用飞机的方式出行。
　　人生中总是要有很多第一次的吧。
　　有母亲陪着，才知道从家到机场的距离，原来是那么长。
　　长到无言。
　　只是我不知道是为什么，习惯了分离，仿佛这个地方也不属于我，只是一站匆匆过客。
　　这之下的生活是多么现实呀，就像道路延伸到远方。
　　无尽的平原，或是再美丽的风景，与我真的有关吗？
　　在一站一站的过客之中，我们终究会认识自己吗?还是只是停留在别人的赞誉和想象之中？
　　天空为大地投下阴影，生活是不是也是一样？

  飞机飞行在崇山峻岭之间
　　我或许本来就不应该预设，我会在飞机上认真地写完作业。
　　或许跨越地域的时刻，就是适合让自己的灵感流淌吧
　　窗外少云，江心洲历历在目。
　　水网交错，一幅典型的江南水乡感觉。
　　或许一个月后，我又会回到这里，只是是一种踏在地面上的踏实的感觉。
　　时间和空间都是错乱而颠倒的，只是要去度过这一段时间。
　　我们与这个世界失联。我们兀自知道自己的存在。
　　机舱吵吵嚷嚷，让人不能平静
　　我不接受去面向过去的解释。
　　异常平稳的、繁忙的京广线。
　　经过的，甚至还没有被人认出的大武汉
　　我们都是被巧合地相遇在了一起
　　缺少一种共同体的形塑。
　　在这个社会上，分层也日益显著
　　我们终究会面对现实，哪怕我们拥有再多理想。
　　或许现在我们还能固执地将自己束缚在一个蚕蛹之中。
　　等待一种破茧成蝶。
　　期望着一种美景。
　　你会发现，人类在这个大地上的痕迹原来是如此明显。
　　我忽然厌倦了打卡拍照的旅游。
　　而人与人之间的感情也变得如此稀薄
　　或许只是找另外一个地方，放空自己。
　　毕竟我们都生活在不同的世界之中，要想找到联系并不容易。
　　我能确定的是自己很好。
　　我们能够跨越时空，但是我们又是否能够真正地跨越，我们自己？
July 8, 2019
