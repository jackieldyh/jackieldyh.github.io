从今又添，一段行路

Posted on 2019-07-07 | In Travel | | Visitors:
Words count in article: 1.6k | Reading time ≈ 5

  我迫不及待的想写下这些文字，因为知道，很多时候，思维的思绪往往就是一闪而过。
　　早上，我们明白了独立和自主的优越性。
　　尽管我们也可能做错很多事情。
　　但是那些都是我们的经历。

  永远在集体之中保持警惕，在独立的时候想着社会。
　　这不是一种分裂，而是一种，人生的，态度或是追求？
　　看向窗外，似乎能让人真的明白，大漠孤烟直。
　　尽管现在还没有落日，明晃晃的太阳照耀着整个大地。
　　一段十个小时的旅程，完全笼罩在阳光之下。
　　或许是要我们进入一段最长的沉睡，去从春季学期的杂事中解脱。
　　毕竟曾经的自己，或许也只有自己知道，辜负了太多太多。
　　还是有一个更好的期望。
　　万米高空，三万英尺的巡航。
　　仿佛已经跨越了一半的地球。
　　遇见的未知的世界让人期待同时令人恐惧。
　　独在异乡为异客。
　　但是并不一定需要思乡。
　　世界终究向我们打开，也终究要我们自己去闯。
　　东西伯利亚山地、中西伯利亚高原、西西伯利亚平原；
　　高加索山、波罗的海、额尔齐斯河。
　　东欧平原，波状起伏。
　　这是我们所跨越的地方，这是我们所梦想的天地。
　　一切似乎是如此现实，我从过来没有见过这么飘渺，而如此美丽的云朵。
　　轻拢慢捻抹复挑。
　　很多时候，美好只属于眼睛。
　　这或许就是所谓的上帝视角。
　　只要我飞的够快，我就能够逃离时间。
　　遇见了一次，就没有然后。
　　穿越波罗的海，似乎对海岸线已经没有了感情。
　　这段旅程的最后10%。
　　想他。

  在慕尼黑机场停留的时间，似乎过得非常快。
　　我真是不能理解，预计18：45起飞的航班，为什么18：25才让人开始登机。
　　妥妥地晚点呀。
　　看来第一次遇见伦敦，我就要看到其22点的模样了。
　　或许可以将东北的很多事情套用在伦敦身上吧，毕竟纬度是那么接近。
　　作为有一种成熟模式的国家和较高国民素质的地方，可能连机上广播都变得更为随意了呢。
　　其实很多事情并不是安全问题，而是规则问题。
　　一个合理的规则才能提高社会的效率，而不是为了1%的更高安全去牺牲更多效率。
　　就比如说我一直在等待着，安全演示？
　　但终于发现这是一种“中国特色”。
　　还有什么是我们习以为常但最终发现是我们的特色的呢？
　　我们应该有一种平视的视角，而不是沉浸在宣传之中。
　　社会的集体疯狂才是更值得警惕的吧。
　　尽管或许我们也会发现，换一种视角，平常十分罕见的事情也是习以为常。
　　但是这才是一个完整的认识呀。
　　也只有先去接受这种完整的认识，才能去实现更为整体的问题。
　　不学礼，无以礼；不学诗，无以言。
　　我们也需要去知道一些，“真正”的正确。
　　哪怕它并不存在，我们永远在追寻的路上。
　　大学就是一个思想的试验场。
　　或许也只是我们为它加上了太多的外部性。
　　Learn to be responsible for yourself.

  然后现在想一想，发现自己竟然是从昨天晚上三点，到今天凌晨三点，整整经过了二十四小时。
　　可以在地球上划出一万公里的二十四小时，还是不常见呢。
　　从思想上的广域到物理上的广域，我们经历着这场自导自演的实验。
　　一场十九岁半的成人礼吧。
　　然后我们深知自己之前时光的虚度，但是还是找不回一年前的状态。
　　靠过去为自己寻找勇气。
　　那么这场过去会否越溯越长？
　　成长总是发生在不知不觉之中，我们却总要为它找到某个标志。
　　伴随着引擎的轰鸣，或许我们又可以说我们又进入了一个新的阶段吧。
　　对时光，永远保持敬畏。
　　日落的向西飞行的航行，永远是被染上色彩的。
　　这个强烈的上升和剧烈的转弯，让人感受到一丝担心，但是又感叹这种强力。
　　就是在这种矛盾之中，我们找寻着对这个世界的认识。
　　几率在没有发生之前，就永远只是数字。
　　我们所看到的现实，又怎么知道是真实？
　　我想我只有去亲身感受，然后去形成自己的经验。
　　那是独属于自己的。

  我们很快的穿越云层——或许这就是小飞机的好处吧。
　　云下一个个小城镇，共同组成了发达国家的城市网。
　　机舱里静地出奇，没有担心熊孩子的烦恼。
　　这也就是独行的好处吧。
　　能够绝对的控制自身。
　　看到八点钟还没有日落的太阳。
　　而有的时候我们又或许是太过于需要在集体之中找到认同，将每一件事在网上择摘地记录。
　　或许只有自己知道，自己所认为的现实。
　　很多时候这是没有办法验证的。
　　就这样，如果你认为如此。

  又或许是计划永远赶不上变化，最终在路边的快餐厅对谈到深夜？
　　逐渐将两个世界拼合，但是又发现自己与对方，终究是两个个体，拥有不同的经历。
　　有一点儿各说各话。
　　但是如今，找到一个倾听者也已经是这么困难。
　　我们将之前的数年，化成一个夜晚。
　　却不得不接受现实。
　　显示的道路总是会分成两条
　　而我喜欢人迹罕至的那条
　　不自己走过又怎么知道精彩与否？
　　谢谢你，给了我单纯，给了我指引。
July 12, 2019
