当生活被按下暂停键

Posted on 2019-11-05 | In Comments | | Visitors:
Words count in article: 1.2k | Reading time ≈ 3
“世界上只有一种英雄主义，那就是得知生活的真相之后仍然勇敢的生活。”
——罗曼·罗兰
   你会突然发现，不论在哪里，冷空气造成的影响似乎都是同步进行的，不论是在大风的北京，或是久违的江南。地球同此冷暖，但每个人的感受却并不会是如此相同、也只有在这个时候，你才能够知道托尔斯泰的那句话原来是如此正确：“幸福的家庭总是相似的，不幸的家庭各有各的不幸。”
　　我想，很难说这真的是某种“不幸”，尽管他会耗费人大把的时光。或许这样的话有一些太“政治正确”，都已经是这样的情况又能改变多少呢？在面对茫茫的现实之后，终究还是要奋然而前行。
　　说到底，谁会想到在那个风和日丽的一天之后，这个学期的时间就会发生这么大的转变呢？但从某个意义上来讲，自己又是不是希望发生这样大的转变呢？从第五走到第九周，自己又究竟干了一些什么呢？一切都是掩饰罢了，一切都是托辞罢了。尽管你会有一个清晰的预期，这种行为肯定能够被理解，甚至还可以维护自己的某种有关“形象”的执念，但是面对真实的自我，又应该如何是好？
　　其实只要多想几步，就会发现时间，或许真的是这个世界上最可宝贵的东西。尽管你可以假装鸵鸟看不到这一切，但是该来的总会在某个时刻来临，在一个黑箱似的环境之中，每个人都被某种不安感所裹挟，这一方面导向着内卷，一方面同样激发着鲶鱼效应。我尽可以选择一条不同的道路，但是在选择道路之前还是需要有一个共同的基础，当人感觉自己的知识越堆越多而自己的问题却越来越少的时候，这其实不应该是一种正常的状态吧。说到底，从进入大学以来，尽管写下了很多文字，但是自己总是在向一个更加“舒服”的地方前进，并且不再想出来。
　　能够想象很多美景，但是当许多东西停留于表面与感官，当许多东西停留于信息，究竟是要“记得”，还是要去“思考”，答案或许非常显然，但作为一个问题，放在不同的环境之下，又是如此的难以交流。
　　我们不可避免的仍然被许多名字所累，对每一个都保有着一种原始的好奇，但却又要仔细的去想这背后究竟蕴含着什么，或许不应当是简单的被一种不可名状的伟大所包绕的感觉。
　　这或许是一种抬头仰望天空所得来的四教，但是地面上总是有太多的事，在围绕着我飞行。
　　当你如此近的感受到死亡的距离，当你的生命在一刹那间被下了判决，当你提前体会到了可能是五十年之后的感受，你又是否有勇气去接受着一个残酷的事实，尽管你的脑袋或许还能清晰的思考，你甚至还能清晰地表达。
　　一整个月地仰望，或许并不能让人得到有关天空的更多信息，但是却能让人得到有关自己的更多信息。或许从来没有从这样一个角度去看向这个世界，但是当时间被停止，当视觉被限制，会发现这种视角却会给人许多不一样的信息。那明晃晃而阴翳的天空，那车水马龙停留于想像，那远方的一切变成某种幻影，会让人愕然，停滞， 而后前行。
　　这里你会遇到形形色色色的人，被一些奇异的关系所联结，你很难说这是合理的，人与人之间根本没有共同体关系，但是却又是如此紧密依靠。如果听惯了民族、民主或是自由的符号，你会发现这一切有多么虚伪，而我们处在这之中又都无法逃离。
　　或许这本身，就是让人看到自己无能无力的时刻。
November 18,2019
