万物背后的永恒

Posted on 2019-05-17 | In Daily | | Visitors:
Words count in article: 1k | Reading time ≈ 3

Sentence

青春盛大而华丽，是人生中最美的芳华。

这段多彩的岁月，被无数文学作品用不同的语言和风格反复咏叹。

Poem

西江月

[宋]苏轼
世事一场大梦，人生几度新凉？
夜来风叶已鸣廊，看取眉头鬓上。
酒贱常愁客少，月明多被无妨。
中秋谁与共孤光，把盏凄然北望。

  一首有些，怎么说，可能是有些超脱的词句。
　　但是不可否认，正是在这种关于时间的轮回性思考之中，我们认识了自身。
　　人生或许只有一次，那么这一次的机会与时光，就应该是更加符合自己心意的。
　　原谅我某种程度上，也是在说“精致的利己主义者”。
　　但是在物质性的追求之外，我们是否应该有一些更为高原的，值得我们去用一生思考的命题？能够有一些值得思考的时光？能够自由的抒发情感，去获得一种心灵的满足的空间？
　　它不一定是快乐的，但是一定是属于个人的。
　　变动不居，时节如流。我们永远是生活在这个世界上的，再高远的梦想与思想，终究还是要回归现实，我们也不得不去接受现实的影响。
　　但是这并不意味着我们不能有自己的主体性。“人是一根会思考的苇草”，苇草随风飘荡，但是在这之中，在诸多的不确定性之外，我们能够反思，能够思考能够向着自己希望的方向去努力。
　　如此，孤独也是一种最美的模样了吧。

Excerpt

　　   政治上的家产制组织在整体上有可能更倾向于定型化的架构，亦或者是更倾向于专断的架构。前者主要盛行于西方，而后者在很大程度上局限于东方。因为在那里暴力的神权统治和家产制军事基础经常被新征服者所篡夺，从而有效地平衡了分权化与占有的自然进程。
　　——韦伯：《支配社会学》,145

　　   我们在这门课中学到了些什么？或许这个问题疑问出来就特别的，“庸俗”，但是我们却可能确实是过早的去看到了一些学术圈和这个社会的现实，或者说，在阅读的时候，自己还是会不由自主的用当代的元素去理解古代的思想，或许也正是在这个过程之中，我们迷失了自我。

Comments

  最近总是不由自主的想要吃一些瓜。
　　或许也是因为太久没有这样写过文字了吧，面对几篇三千字的论文，竟是会有一种“身体被掏空”的感觉
　　我不相信这是凤凰涅槃的前兆。
　　或许在这种高压之下，是告诉我们应该去追寻更多，还是要更为努力的提升自己？
　　毕竟细细的算起来，一年的时间就这样悄然过去。
　　我们在这条道路上，终究活成了别人口中的“前辈”
　　但是只有自己才会知道自己是什么模样
　　在鲜花与掌声之中脱光衣服面对自己的不完美
　　保持谦卑，保持快乐
　　生命的路有很多条
　　优秀的定义也不止一种
　　“非典型优秀”的前提，是快乐的。
　　我们学习的理论永远是为我们的世界增添更多的面向，让我们能够在更多的世界中去栖息
　　去能够在这个世界中辗转腾挪
　　去能够在摘下假面之后保持自我
　　去在他人的期望和自己的愿望之间保持平衡
　　去学会接受时间洪流的冲刷，并且借力冲浪
　　让自己努力到无能为力。
　　It is too simple; it is too hard.
