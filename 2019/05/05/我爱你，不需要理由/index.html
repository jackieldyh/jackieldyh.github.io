我爱你，不需要理由

Posted on 2019-05-05 | In Comments | | Visitors:

　　本文为新雅82班2019春主题团日材料，《亲爱的》影片观后感&Pre稿
　　“我知道这个世界我已无处容身，可是，你凭什么审判我的灵魂。”
　　《亲爱的》是2014年上映的，由陈可辛导演的一部有关家庭伦理和人际关系的电影。影片全长130分钟，片中触碰了单亲家庭、拐卖儿童、城乡差距等许多主题。这可以从这部影片其他的名字：”亲爱的小孩 / 打拐 / 家之亲爱的小孩 / Dearest“看出一斑。或许每个主题单独拎出来，都是对这个社会某个方面的反思。但是我们不禁要问的是，将这么多的主体拼接到一起，是否在一种充沛的情感基础上的导向？尽管电影提供了一个开放性的结尾，但是当这些主题混杂在一起时，电影就拥有了社会的复杂——但是电影不应该是一种对生活的升华么？
　　这部影片是有现实世界的真实案例的。总体上，这部电影希望讨论的一个核心主题，或者说矛盾，是情与理的冲突。可以说，大致上，或者说平常意义上看，除了那个最初拐走孩子的那个男人，其他人都没有错。但是实际上，从情感上来看，或许每个人都能够给出自己的判断，但是，我并不希望这部影片，变成一部道德评判，或者说，比惨的大会。
　　简要的来摘取豆瓣的剧情简介…
　　田文军（黄渤 饰）和鲁晓娟（郝蕾 饰）曾是一对恩爱的夫妻，然而，两人之间的感情却被时间和争吵消耗殆尽，最终，他们选择了离婚。如今，联系着两人的唯一枢纽，就是可爱的儿子田鹏。然而，某一天，这唯一的纽带也断裂了，田鹏于一次外出玩耍时无故失踪，绝望和崩溃之中，田文军与鲁晓娟踏 上了漫漫寻子之路，并在途中结识了许多和他们一样无助的父亲和母亲们。
　　时光匆匆流逝，一条关于田鹏的线索浮出水面，促使田文军和妻子来到了一处偏僻的村落之中，在那里，他们看到了酷似田鹏的男孩，然而，男孩口中的“妈妈”却并非鲁晓娟，而是一位名叫李红琴（赵薇 饰）的村妇，这错位的一切究竟是怎么回事？
