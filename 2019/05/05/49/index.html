东南形胜，三吴都会

发表于 2019-05-05 更新于 2021-01-26 分类于 Travel 阅读次数： Valine：
本文字数： 1.4k 阅读时长 ≈ 1 分钟

  从井冈山到杭州
　　九个小时的旅程，却是从建党的最初，到了从起点到发展的历程
　　

  “钱塘自古繁华，有三秋桂子，十里荷花”
　　《望海潮》中的词句言犹在耳
　　终究有一种面相，名为真实。

  当列车驶过钱塘江大桥的时候，我默默地看着这宽阔的江面
　　这或许是我与这座城市相遇的第一面
　　也是中国桥梁发展史上，浓墨重彩的一笔
　　一声长笛，仿佛是这座城市的欢迎
　　在地下穿过这座城市的大街小巷。

  初到就是一座具有浓郁仿古风情的近代建筑
　　旅游区令人作呕的存在。
　　我或许还是更希望走进背街小巷去发现这座城市生活的真实
　　——尽管我注定只是一个过客。

  上城区，吴山广场
　　去一座城市总是要去感受这座城市的博物馆
　　真实的力量，也来自时间的沉淀。
　　登上矗立的亭阁
　　与西湖的第一眼竟是通过这种方式
　　一片湖，与一座城

  默默地穿越整座吴山，从张岱的书屋，到静默的烈士纪念馆
　　 再向下，就真的是西湖了
　　 旁边有很多优质的美术馆和博物馆，在一条香樟簇拥的道路旁
　　 这才是江南呀。

  因为是五一假期，能够预见西湖的人流肯定会非常多
　　 不论是灵隐寺，还是雷峰塔，在自行车的车轮旁，我想他们只是成为了一个地名
　　 ——毕竟对所谓的传说和宗教并不感兴趣。

  但是苏堤和白堤还是要用脚步丈量的
　　 哪怕开头的几步人潮汹涌，到后来你还是会发现
　　 心宽天地宽
　　 看着太阳一点一点的藏到山后，看着人流涌动，流进又流出
　　 这座城市的活力， 在这之中，通过电波，能让我感受，还是有熟悉的人。

  去感受光影变幻之下的绚烂
　　 从西子湖畔，到展览馆前
　　 很多时候，建筑的宏大只是带来了体量上的改变，但是真正的让人说结婚瘦的空间，是在规则之下的人情味，是最平平淡淡的小笼包、煎饼和蒸饺
　　 一座有生活气息的城市，不应该只有规则和金钱。
　　 在广阔而沉静的江面上，甚至没有一丝的风，但是也能够感受到， 这座城市的分裂与拼合
　　 毕竟，我们面对的，是不完全了解，或许也不能够完全了解的世界。
　　 我不是归人，是个过客。

   第二天就更加地匆匆了，从西湖博物馆到浙江省博物馆，脚步丈量的，是高度凝练的信息，或者说知识。
　　 某种程度上，我甚至有些享受这种能够一个人思考的时光。
　　 总是需要自己的空间来放空自己，不需要在，哪怕只有两个人的关系之中伪装。
　　 很多时候，我们太过急于的追求认可，反而失去了对自己行走方向的自信。
　　 我们身处于关系之中，无往不在
　　 但我们也终究属于我们自己，无法改变

   坐上公交车，像从雅典的宪法广场去拜访柏拉图学园
　　 浙大紫金港校区
　　 是这趟旅程中为数不多的，有关于人的因素了吧
　　 而倘若有人，景便不再是景，而是人化的景象
　　 哪怕没有多少风景，静静的陪伴也是一种享受。
　　 这种穿越的时间、和空间的对话，总是能让人从当下的世界中抽离出来
　　 尽管我们所能拥有的时间是如此宝贵
　　 甚至都没有让人准备好如何表达情感
　　 回忆过去，且笑且歌
　　 未来在过去之上
　　 但彼此分离，总是回不到过去的模样

   穿越大半个杭州，这种感动仍然清晰，历历在目
　　 高速的车流开始拥挤，中国人口最多、经济最发达的地区，从表明上看，与发达国家并无二致
　　 但是这就是我们充满复杂性的土地
　　 这就是我们所要面对的现实
　　 不求理解，只求生存
　　 在机场的四五个小时，是一种沉淀？还是一种掺杂着自豪、快乐与焦躁的时光？
　　 我终于看到了凌晨两点半的北京城。
May 11-12, 2019
