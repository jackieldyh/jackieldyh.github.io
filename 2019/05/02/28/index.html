有幸平生志，天涯共此识

发表于 2019-05-02 更新于 2021-01-26 分类于 Comments 阅读次数： Valine：
本文字数： 947 阅读时长 ≈ 1 分钟

本文为新雅书院2019年学生节第二篇推送文案（初稿）

半亩方塘一鉴开

一方天光折射出对未来的渴望
一枝彩铅勾勒出对现实的期许
一张书页记录着思想的闪光点
人生在知识海洋中泅渡
有界又无疆

而我们相知，相遇，相识
院儿给共同体提供场域
你听
但余钟磬音

当现代性的洪流瓦解差序格局
当功利性的浪潮度量世间关系
当世俗化的追求凝滞逐梦步伐
漠然地随波逐流
被环境消磨棱角

而我们体验、创造、思考
院儿给思想提供实验室
你看
窗含西岭千秋雪

山峦为晴雪所洗
娟然如拭
鲜妍明媚

从院儿出发
追寻一方院儿
时光里流淌的故事
发生在一个个无比精彩而奥义万千的
院儿里

有源儿
日出而落日落而息
喂马劈柴周游世界
车灯漂移汇成长河
生命之河汹涌而盛大
生命之源清冽而潺潺

从“不忘初心方得始终”
到“前事不忘后事之师”
谈论布特鲁、施特劳斯
也要明白涂尔干、黑格尔
更不用说上下千年，
从百家争鸣到宋明理学

如果所有的成功都有一个最微茫的开始
那又如何解释，
人类文明最初的思想火花又是如此炫目而瑰丽。

有缘儿
在无数可能中找寻确定
相遇在光年外的航标
是神秘的相互作用
让彼此紧紧相拥
汇聚在这一方
开放的天地
说出姓名
推开门
相遇

相遇
又逡巡
荷塘月色
紫藤萝瀑布
学堂路的倩影
拼成五彩的画面
汇成青春的万花筒
向我们给出无数可能
每片碎片都是青春模样

有垣儿
垣墙周庭
每人都有每个人的热闹
组成了快乐的湖沼
快乐是能够传染的
也是能够溢出的

快乐或许是DDL的随后一刻的提交
或许是上课时的灵魂对话
或许是运动场上放飞的自我
奋斗的青春总有不同的模样
在一起的相识，才最为珍贵

有苑儿
春天的院子里总是能开满繁华
键盘声像燕在林间呢喃
少不了蓝色多瑙河的静谧
也能看到天鹅湖的翩跹

一点似能闻出茉莉芳香
一横也能梦回长城巍峨
在心的远景里
一切都会是最好的模样

你说能不能换种文风？
有鸳儿
友院儿
幼院儿

院者，周垣也。
最初的最初，
院儿只是一圈篱墙，
聚拢了散落的人，
也聚拢了游离的心

随后的时日里，世界千百般变幻，
院儿不只是四方篱墙。
它不再只是圈住的土地，
而是那些相会一处的灵魂。

院儿是围炉拥火，
院儿是曲水流觞，
院儿是许多人的温暖。
院儿是许多心的共鸣。

因为有缘儿，人们相聚院中；
因为有院儿，我们生逢其时。

May 4, 2019
